Abstract

Neste trabalho com um XSLT aplicado ao Schema dos júris são geradas as queries para a criação das tabelas da base de dados.

Depois com um script de PHP são gerados os inserts a partir do ficheiro XML com informação dos júris.

De notar que a base de dados foi construida na 3º forma normal, o que dificultou a criação de um XSLT genérico
