Abstract

Neste projecto foram desenvolvidas XSLT para um SIP.

XSL ou EXtensible Sytlesheet Language é uma linguagem que permite desenvolver transformações (XSLT) para ficheiros XML que nos permita transformá-los de forma a obter outros XML ou outros objectos tais como páginas HTML como é o caso deste projecto.

Para além das XSLT já referidas, este projecto contempla ainda uma Document Type Definition, ou simplesmente DTD, assim como um XML onde se aplicam as regras da mesma e que é usado como base deste relatório.

De forma a tornar a experiência mais agradável na visualização do resultado das XSLT foram também inseridas no projecto regras de estilo, através de CSS, assim como as fotografias dos membros do grupo de trabalho.
